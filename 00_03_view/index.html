メモリ管理がやばそうじゃん

Viewのインスタンスを生成し続けるとviewとmodelはメモリに残り続ける。
とすると下記のrouterが問題ありそう。

router.js page01: function () {
　 new SimpleView();
},
page02: function () {
　 new SimpleView();
},
page03: function () {
　 new SimpleView();
}

インスタンスを作る前に、前に作ったViewとModelを削除する必要がある。
たとえば、

router.js page01: function () {
　 instanse.clear();
　 instanse.start();
}
